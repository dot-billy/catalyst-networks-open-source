Node Management CLI

To add new nodes or lighthouse nodes to your organization, use the {{ project_meta.NAME }} Management CLI. This approach is more secure and flexible than using the dashboard. Follow the steps below for your operating system.

Download the {{ project_meta.NAME }} CLI:
  • Linux (x86_64)
  • macOS (Intel)
  • macOS (Apple Silicon)
  • Windows (x86_64)
  1. Download the CLI binary:
wget https://catalyst-networks.nyc3.digitaloceanspaces.com/go-binaries/catalyst-networks-linux-amd64
chmod +x catalyst-networks-linux-amd64
sudo mv catalyst-networks-linux-amd64 /usr/local/bin/catalyst-networks
  1. Run the following command to add a node:
catalyst-networks create --name mynode --hostname mynode.example.com
  1. To add a lighthouse node:
catalyst-networks create --name mylighthouse --hostname mylighthouse.example.com --lighthouse
  1. Download the CLI binary:
curl -LO https://catalyst-networks.nyc3.digitaloceanspaces.com/go-binaries/catalyst-networks-darwin-amd64
chmod +x catalyst-networks-darwin-amd64
sudo mv catalyst-networks-darwin-amd64 /usr/local/bin/catalyst-networks
  1. Run the following command to add a node:
catalyst-networks create --name mynode --hostname mynode.example.com
  1. To add a lighthouse node:
catalyst-networks create --name mylighthouse --hostname mylighthouse.example.com --lighthouse
  1. Download the CLI binary:
Invoke-WebRequest -Uri `
  "https://catalyst-networks.nyc3.digitaloceanspaces.com/go-binaries/catalyst-networks-windows-amd64.exe" `
  -OutFile "catalyst-networks.exe"
  1. Run the following command to add a node:
.\catalyst-networks.exe create --name mynode --hostname mynode.example.com
  1. To add a lighthouse node:
.\catalyst-networks.exe create --name mylighthouse --hostname mylighthouse.example.com --lighthouse
Tip: You can generate registration tokens in the "Registration Tokens" section of your organization. The CLI will prompt you for the token and other credentials when you run the create command.
